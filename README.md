## Description

Scans a given URL for vulnerabilities in the request headers and cookies

## Use

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the requirements.

```bash
pip install -r requirements.txt
```

Run the python file

```bash
python app.py
```
